Docker Piwik
============

Piwik is the leading Free/Libre open analytics platform.

Installation
--------------

```sh
git clone https://github.com/jcherqui/docker-piwik
docker build -t piwik .
```

Quick Start
-----------

`docker run -p 80:80 piwik`

Point your browser to `http://127.0.0.1`
